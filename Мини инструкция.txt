### TOKEN: you_token - помещаем ваше токен бота.

### ADMINCHAT - you_adminchat помещаем ID вашего чата для администраторов
### обязательно участники чата должны быть админами или Супер админами в базе данных
### добавить администраторов можно через админ меню

### SUPERADMIN: tg_id - Ваш id телеграмм аккаутна, 
### user_name - Любой ник для отображения в бд, так же можно будет изменить через админку
### right - Оставляем 'Супер администратор' для полных прав

### CATEGORY: Прописываем желаемые категории в виде ['Продукты','Одежда','Другое']
### Служит неким ограничением для продавцов, они выбирают только те, которые тут узаканы

{
	"TOKEN": "you_token",
	"ADMINCHAT" : "you_adminchat",
	"SUPERADMIN" : "{'tg_id':'you_id','name':'user_name','right':'Супер администратор'}",
	"CATEGORY" : "{['Продукты','Одежда','Другое']}" 
}
